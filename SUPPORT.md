# Support

Use GitHub Issues for reproducible bugs, feature proposals, and documentation
gaps:

<https://github.com/chetto1983/turing_AgentMemory_MCP/issues>

Before opening an issue:

1. Read the [documentation index](docs/README.md) and
   [known limitations](docs/limitations.md).
2. Check `/health` and record content-free stage identities.
3. Reduce the problem to synthetic data.
4. Remove credentials, memory text, documents, local paths, and user IDs.
5. Include commit, OS, Python or image version, provider model identities,
   vector dimensions, and exact commands.

Use the private process in [SECURITY.md](SECURITY.md) for vulnerabilities. Do
not use public issues for customer data or secrets.

This community project does not currently offer a paid support SLA.
