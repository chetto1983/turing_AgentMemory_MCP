| untrusted document text -> Entity vertices + MENTIONS edges | Extracted entity names from arbitrary document content become graph nodes and traversal seeds. |
| shared ArcadeDB tenant database -> chunk-entity traversal | A traversal crossing an entity could surface another tenant's chunk if any hop loses its scoping. |
